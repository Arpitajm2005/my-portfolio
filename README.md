# my-portfolio
# 🌐 My Portfolio Website

A modern and responsive **personal portfolio website** built using HTML, CSS, and JavaScript.
This project showcases my skills, projects, and contact information with a clean **glassmorphism UI** and animated background.

---

## 🚀 Live Demo

🔗 https://arpitajm2005.github.io/my-portfolio/

---

## ✨ Features

* 💎 Modern UI with **Glassmorphism effect**
* 🌌 Animated star background
* 🌙 Dark / Light mode toggle
* 📱 Fully responsive design
* 🧾 Projects showcase section
* 📩 Contact form UI
* ⚡ Smooth hover animations

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/Arpitajm2005/my-portfolio.git
```

2. Open the folder:

```
cd my-portfolio
```

3. Run the project:

* Open `index.html` in your browser

---

## 🌐 Deployment

This project is deployed using **GitHub Pages**:

1. Go to repository **Settings**
2. Click on **Pages**
3. Select branch: `main`
4. Save and get your live link
